### Como Usar Cada Seção

#### 1. ✅ Tarefas
```
1. Digite uma tarefa em "Adicionar nova tarefa..."
2. Selecione prioridade (normal/alta/baixa)
3. Clique "+ adicionar"
4. Clique no checkbox para marcar como concluída
5. Clique no X para deletar
6. Use filtros para ver: todas/pendentes/concluídas
```

#### 2. 💰 Finanças
```
1. Digite descrição (ex: "Compras")
2. Digite valor (ex: 50.00)
3. Selecione tipo (receita/despesa)
4. Selecione categoria (alimentação, etc)
5. Clique "+ adicionar"
6. Veja os gráficos atualizarem automaticamente
7. Histórico de transações abaixo
```

#### 3. 📝 Notas
```
1. Clique "Nova Nota"
2. Digite título (opcional)
3. Digite conteúdo (auto-salva)
4. Clique em outra nota para ver lista
5. Clique X para deletar
```

#### 4. 👤 Perfil
```
1. Veja informações do seu usuário
2. Histórico de uso (Última coluna mostra quanto tempo passou em cada página)
3. Estatísticas de uso
4. Páginas mais visitadas
5. Opção para limpar histórico
```

#### 5. ⚙️ Admin (você é admin)
```
1. Veja usuários registrados
2. Veja logins de hoje
3. Veja total de logins
4. Veja últimos logins com horário e status
5. Clique "limpar logs" para deletar histórico
6. Clique "banir" em usuário para banir
7. Ver usuários banidos
```
