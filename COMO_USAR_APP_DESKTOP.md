## 🔧 Solução de Problemas

### O app não abre?
```
1. Verifique se o arquivo Aphaday.hta existe
2. Clique 2x nele
3. Windows vai abrir como programa
```

### Aparecer aviso de segurança?
```
Clique "Executar" - é seguro!
O aviso aparece porque é um arquivo .hta
```

### Dados desapareceram?
```
Limpe o cache do navegador:
1. Ctrl + Shift + Delete
2. Selecione "Apagar tudo"
3. Recarregue (F5)
```
